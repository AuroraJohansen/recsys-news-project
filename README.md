# News Recommendation System

Project for TDT4215 Recommender Systems.

## Goal
Build and evaluate recommendation models for news articles using the EB-NeRD dataset.

## Planned Methods
- Baseline (most popular / most recent)
- Content-based filtering
- Collaborative filtering
- Hybrid models

## Evaluation
Metrics:
- Precision@K
- Recall@K
- NDCG@K
- MRR

## Repository Structure
- `src/` implementation
- `notebooks/` experiments
- `scripts/` runnable pipelines
- `configs/` experiment configs
